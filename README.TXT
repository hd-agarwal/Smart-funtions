PROJECT TITLE: Smart Functions
PURPOSE OF PROJECT:An interactive terminal-based program that can perform multiple mathematical and logical operations;
                    also involves a menu-system driven approach to make user experience smooth.
VERSION: 1
HOW TO START THIS PROJECT: From Bluej IDE for Java 
                            1) Open the directory as a Bluej project
                            2) Compile all the classes using the compile button
                            3) Right click the Mega class and run its main method
                            (How to install Bluej: https://youtu.be/OHdZJZETPFQ)
                           
                           To run without BlueJ(Command-line)
                            1) Navigate to the project folder and compile the Mega class using "javac Mega.java" command(without quotes)
                            2) Run the program using "java Mega" command(without quotes)
